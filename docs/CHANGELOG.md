# 변경 이력

## Final UX 보완

- 사용자 지정 우선 재료를 활용하는 메뉴를 독립된 최우선 추천 그룹으로 분리
- 홈의 오늘 먼저 확인 목록에서 사용자 지정 재료를 자동 우선도보다 앞에 표시
- 별도 식사 기록 메뉴와 화면 제거
- 조리 완료 시 재료 사용량만 입력하고 같은 화면에서 재고 반영 완료 카드 표시
- 같은 식재료 구매 묶음을 화면에서 구분할 수 있도록 재고 ID 표시

## Final

- 정확 재료 우선·동등 대체 후순위의 순서 독립 매칭 적용
- 크림 파스타, 버섯 리소토, 크림 감자수프 회귀 테스트 추가
- 재고·양념 변경 시 기존 추천 결과 자동 무효화
- 조리 완료 API에 레시피 문맥 기반 대체 검증과 최근 추천 확인 추가
- 추천 폼 기본 조건과 상세 설정 분리
- 최종 제품 중심 README와 문서 정리
- 레시피 source 값을 `curated_internal_seed`로 통일
- 캐시·테스트 DB 제거, MIT License 추가

## 이전 주요 개선

- 저장형 냉장고와 SQLite 영속성
- 식품군별 상태 질문과 표시기한 없는 재료 관리 구간
- 추천 결과 그룹 분리와 추가 재료 안내
- 대체 조리기구와 문맥 제한 대체 규칙
- 조리 완료 후 재고 차감과 식사 기록
- 냉장고 콘셉트 홈 화면
